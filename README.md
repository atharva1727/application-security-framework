# 📌 Overview

This project implements security mechanisms for a university web application using server-side validation, secure form handling, and session management techniques.

The goal is to reduce vulnerabilities and prevent unauthorized data entry.

---

# 🎯 Features

✅ Input validation

✅ Whitelisting & blacklisting

✅ Registration form security

✅ Password verification

✅ Session management

✅ Secure authentication workflow
